# 💸 AI Expense Tracker

A modern React-based expense tracker with smart insights and currency conversion.

## 🚀 Features
- Add & delete expenses
- Category-wise breakdown
- Real-time currency conversion
- Smart insights based on spending
- Local storage persistence
- Responsive UI

## 🛠 Tech Stack
- React (Vite)
- JavaScript
- CSS
- Exchange Rate API
